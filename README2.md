## 🌿 關於創作者 (About Me)

* **創作者**：梔久 (Zhijiu)

* **開發裝備**：MacBook Pro M3 / VS Code / GitHub Desktop

## ✨ 專案特色 (Features)

- [x] **日系極簡美學**：採用米白、霧棕與柔和淡灰配色，呈現溫暖無壓力的視覺體驗。
- [x] **響應式排版**：選用卡片式網格系統（Grid System），完美適配 Mac 與行動裝置瀏覽器。
- [x] **AI 協同開發**：深度運用 Prompt Engineering（系統提示詞與 Few-Shot 技巧）指導生成高效能程式碼。
